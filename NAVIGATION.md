## 🎨 Personnalisation facile

### Ajouter un 3ème article :
1. Copie `article-template.html`
2. Renomme (ex: `ethereum-analyse.html`)
3. Remplace le contenu
4. Dans `index.html`, change le lien de l'Article 3

### Changer les stats :
Dans `index.html`, ligne ~300 :
```html
<div class="stat-number">6+</div>  ← Change ici
<div class="stat-label">Années d'expérience</div>
```

### Ajouter un réseau social :
Dans `index.html`, section social (~ligne 340), ajoute :
```html
<a href="TON_LIEN" class="social-card">
    <span class="social-icon">📱</span>
    <span>Telegram</span>
</a>
```
