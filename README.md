# 🚀 HarK Trading - Site Web

Ton site web personnel pour tes analyses crypto !

## 📁 Structure du site

```
site_hark/
├── index.html              → Page d'accueil
├── solana-2026-2027.html  → Article Solana (complet avec images)
└── article-template.html   → Template pour nouveaux articles
```

## 🎯 Comment utiliser

### 1. Ouvrir le site localement
Double-clique sur `index.html` pour voir ton site dans le navigateur.

### 2. Ajouter un nouvel article
1. Copie `article-template.html`
2. Renomme-le (ex: `bitcoin-analyse.html`)
3. Modifie le contenu :
   - Titre
   - Date
   - Texte
   - Ajoute tes images
4. Ajoute une carte pour cet article dans `index.html` (section Articles)

### 3. Personnaliser la page d'accueil
Dans `index.html`, tu peux modifier :
- **Stats** (ligne 200) : Change les chiffres
- **Articles** (ligne 230) : Ajoute/modifie les cartes articles
- **Réseaux sociaux** (ligne 320) : Change les liens

## 🌐 Mettre en ligne (GRATUIT)

### Option 1 : Netlify (RECOMMANDÉ)
1. Va sur https://netlify.com
2. Drag & drop le dossier `site_hark`
3. Ton site est en ligne !
4. URL gratuite : `ton-nom.netlify.app`

### Option 2 : GitHub Pages
1. Crée un compte GitHub
2. Upload le dossier
3. Active GitHub Pages dans les settings
4. URL : `ton-nom.github.io`

### Option 3 : Vercel
1. Va sur https://vercel.com
2. Import le dossier
3. Deploy automatique

## 🎨 Personnalisation des couleurs

Dans chaque fichier HTML, cherche ces couleurs dans le `<style>` :
- `#00ff9d` → Vert néon (couleur principale)
- `#8a2be2` → Violet (couleur secondaire)
- `#000` → Noir (fond)

Remplace-les par tes couleurs préférées !
